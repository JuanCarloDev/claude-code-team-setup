# CLAUDE.md Template for Penguin Team Members

> Copy this file to `~/.claude/CLAUDE.md` on your machine and fill in your details.
> Claude Code reads this file EVERY TIME you start a session — it's how it knows who you are.

---

```markdown
# CLAUDE.md — [Your Name]

## Who I Am

- **Name:** [Your full name]
- **Role:** [Your role at Penguin — e.g., CEO, PM, Developer, Social Media]
- **Experience with code:** [None / Beginner / Intermediate / Advanced]

## My Projects

- [List the projects you work on, e.g.:]
- Penguin Software (Master Brain)
- Penguin Method (Shopify theme)
- Penguin Wellness CBD (Shopify theme)
- Penguin Quiz

## My Common Tasks

- [List what you typically do, e.g.:]
- Edit Shopify themes (design, layout, sections)
- SEO setup (Google Analytics, Tag Manager)
- Content management (blog posts, product descriptions)
- Code development (features, bug fixes)

## How to Help Me

- [Adjust based on your experience level:]
- Explain git operations simply before executing them
- Always confirm before pushing code to GitHub
- Before making destructive changes (deleting files, force-pushing), ask me first
- When I say "save this", create a document in the project's docs/ folder
- Before ending a session, remind me to save a summary of what we did

## Communication

- **Language:** English
- **Style:** Direct and clear, no unnecessary jargon
- **When uncertain:** Ask me before proceeding

## Git Rules

- Always pull latest code before starting work
- Use conventional commits: feat:, fix:, docs:, chore:
- Never force push to main
- For big changes, create a feature branch

## Important Context

- Our GitHub: https://github.com/JuanCarloDev
- Team setup: https://github.com/JuanCarloDev/claude-code-team-setup
- Always use Opus model with max effort and extended thinking
```

---

## How to Install

1. Copy the content above (between the ``` markers)
2. Create the file `~/.claude/CLAUDE.md` on your machine
3. Fill in your details
4. That's it — Claude Code will read it automatically every session

Or ask Claude Code to do it for you:

```
Hey Claude Code, create a CLAUDE.md file for me at ~/.claude/CLAUDE.md
Here's my info: [paste your details]
```
